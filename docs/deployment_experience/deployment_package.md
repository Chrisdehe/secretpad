## 平台简介

隐语平台部署包，可快速帮助用户体验隐语核心能力。在部署安装包的时候，可选择不同的部署模式，如部署成 *
*P2P的模式（无需可信第三方，无Center平台**
或者 **中心化的模式（有Center平台）;**

中心化模式下，体验版内置Alice和Bob两个节点，配置相应数据，用户可直接进行模型训练；用户也可以自己添加数据表，上传自有数据进行模型训练，数据均在本地保存，无需联网即可体验。
